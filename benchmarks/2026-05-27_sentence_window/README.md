## 关键发现

1. **全面退化**：所有检索模式、几乎所有指标低于 05-23 层级索引基线
2. **单句嵌入语义太弱**：Sentence Window 适合 factoid QA（答案在 1-2 句），不适合心理学领域知识检索（答案分散在多句子中，需要语义词共现）
3. **Recall@1 从 0.29 跌至 0.21**：单句匹配的信号远弱于 500 字段落多词共现
4. **Answer Relevance 降至 0.48**：窗口文本（5 句 ~300 字）虽然比单句好，但仍不如完整的 500 字段落
5. **BM25 退化严重**：单句的关键词密度远低于长段落

### 核心结论

**Sentence Window 不适用于当前领域**。心理学知识的概念（如"DSM-5 诊断标准"）需要多句话共同描述，单句嵌入无法捕捉这种跨句语义。
