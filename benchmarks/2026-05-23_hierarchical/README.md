## 逐模式详解

### Vector (纯语义检索)

```
指标         基线      层级索引    变化
─────────────────────────────────────────
recall@1    0.413     0.293    ⚠️ -29%
recall@3    0.472     0.446    ➡️ -6%
recall@5    0.514     0.544    ✅ +6%
recall@8    0.568     0.669    ✅ +18%
precision@5 0.293     0.387    ✅ +32%
mrr         0.330     0.494    ✅✅ +50%
hit@5       0.467     0.667    ✅✅ +43%
map         0.232     0.370    ✅ +60%
```

**解读**: recall@1 下降是因为层级摘要偏宏观，不如精确段落匹配 #1 位；但 recall@5/8 和 MRR/Hit@5 大幅提升，说明摘要丰富了候选池，相关内容在 2-8 位被找到。

### Hybrid (RRF 融合)

```
指标         基线      层级索引    变化
─────────────────────────────────────────
```

**解读**: Recall 基本持平，MRR 和 Hit@5 显著提升。Hybrid 模式受 BM25 拖累但在排序（MRR）和命中率（Hit）上仍有增益。

### 生成质量

```
指标              基线      层级索引    变化
─────────────────────────────────────────
Faithfulness      1.000     0.987    ➡️ -1%
Answer Relevance  0.527     0.573    ✅ +9%
```

**解读**: Faithfulness 几乎不变（层级摘要本身基于原文，忠实度高）。Answer Relevance 提升 9%，说明摘要提供了更全局的上下文，使回答更全面贴切。

---

## 关键发现

1. **排序质量大幅提升**：vector MRR +50%，说明层级摘要帮助相关内容排到更靠前的位置
2. **命中率大幅提升**：Hit@5 +43%，更多用例能在 top-5 中找到相关文档
3. **BM25 受摘要稀释**：短摘要不如长段落在关键词匹配上有优势
4. **生成答案更相关**：Answer Relevance +9%，层级摘要提供了更完整的上下文
5. **Faithfulness 不受影响**：摘要基于原文生成，忠实度保持高位
